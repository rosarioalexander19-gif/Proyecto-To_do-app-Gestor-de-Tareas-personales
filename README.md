# Gestor de tareas personales.

**Descripción:**

Como lo dice su nombre, es un gestor de tareas organizada por proyectos, donde en cada proyecto permite agregar un sinnúmero de tareas ilimitadas. La idea de este proyecto es que el usuario pueda organizar mejor sus tareas o actividades, las cuales se separarán en proyectos y se podrán visualizar en una lista. 

En este gestor se puede crear proyectos y, en cada uno, se pueden añadir distintas tareas, las cuales se pueden marcar si están realizadas o no. Además, el usuario tendrá la facilidad de actualizar cualquier información dentro de los proyectos y tareas. También se pueden eliminar proyectos o tareas que ya no se necesiten. Por lo tanto, es un gestor que mantendrá todas las actividades del usuario al día.

**Características:**

● Crear proyectos
● Crear tareas
● Visualizar proyectos con sus tareas
● Actualizar proyecto o tareas cambiando la información
● Eliminar proyectos y tareas

**Instalación:**

Para ejecutar este proyecto en tu computadora, sigue los siguientes pasos:
1.	Descarga o clona el repositorio desde GitHub en tu computadora. 
2.	Abre la carpeta del proyecto usando Visual Studio Code. 
3.	Una vez abierto el proyecto, busca el archivo principal llamado main.py. 
4.	Abre el archivo main.py dentro de Visual Studio Code. 
5.	Presiona el botón de ejecutar que aparece en la parte superior derecha con forma de triángulo. 
6.	Automáticamente se abrirá la ventana del gestor de tareas hecho en Tkinter.

**Nota:** Este proyecto está desarrollado con el lenguaje de programación Python. Si no lo tiene descargado, instálelo.
Para que la app le funcione correctamente, al momento de crear una tarea o proyecto se debe llenar los campos que pide, ya que si este está vacío, la tarea o proyecto no se crearán.
